# WebsitePal Landing

Static landing page for `websitepal.pro`.

## Files

- `index.html` is the page markup and production metadata.
- `styles.css` contains the extracted landing-page CSS.
- `script.js` contains the gallery, process-card, reveal, and sticky-CTA behavior.
- `assets/` contains the local logo, favicon, hero image, and website examples.

## Vercel

Create a new Vercel project using this folder as the project root.

- Framework preset: Other
- Build command: leave blank
- Output directory: `.`
- Production domain: `websitepal.pro`
- Optional redirect domain: `www.websitepal.pro`

The primary CTA links point to:

`https://portal.websitepal.pro/sign-up`
